# Day 3 - Linux Navigation and Log Discovery
## Commands practiced
- pwd
- ls
- ls -la
- cd /
- cd /home
- cd ~
- find /etc -name ''*.conf''
- grep ''root'' /etc/passwd
## What I Learned
- pwd displays the current working directory.
- ls lists files and folders.
- cd changes directories.
- find searches for files.
- grep searches for text within files.
- configuration files usually have the .conf extension.
- The root account has UID 0.
## Challenges
- Encountered ''Permission Denied'' when accessing some directories.
- Learned that some files require elevated privileges.

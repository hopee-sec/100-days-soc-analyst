# Day 2 - Linux Permissions & Basic Commands 
## What I Learned
Today i practiced basic linux commands and understood how file permissions work.
## Commands practiced
- ls -l -> view file permissions
- chmod -> change permissions
- sudo -> run commands with admin privileges
- chown -> change file ownership
- mkdir -> create directories
- touch -> create files
- cd -> navigate directories
## Key concepts
- rwx = read, write, execute
- 600 = private file (owner only)
- 644 = owner can edit, others can read
- 755 = scripts can be executed by everyone except edits
## What I now understand
File permissions control access to files and are important in cybersecurity because misconfigured permissions can expose sensitive data or allow unauthorised execution

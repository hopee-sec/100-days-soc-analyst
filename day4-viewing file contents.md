# Day 4 - cat, head, tail and less
## Objectives
- Learn how to view and navigate file contents in Linux
## Commands Learned
### cat
#### Displays the entire content of a file
- Example: cat sample.txt
### head
#### Displays the first 10 lines of a file and can be set to preference
- Example: head -5 sample.txt which will display the first 5 lines of a file
### tail
#### Displays the last 10 lines of a file and can be set to preference
- Example: tail -3 sample.txt which will display the last 3 lines of a file
### less
#### Allows you to view and scroll through a file page by page
- Example: less sample.txt
## What I Practiced
- Created a file named sample.txt using Nano
- Added 10 lines of text to the file
- Viewed the entire file using cat
- Displayed the first 5 lines using head
- Displayed the last 3 lines using tail
- Opened and navigated the file using less
## Key Takeaway
 The commands cat, head, tail, and less are useful for viewing file contents. they help Linux users and cybersecurity professsionals inspect files and logs efficiently.
